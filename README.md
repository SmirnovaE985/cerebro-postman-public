# Cerebro Postman Public Collection

Публичный набор Postman-коллекций и example environment для демонстрации и тестирования API-сценариев.

## Структура проекта

- `collections/` — экспортированные Postman-коллекции
- `environments/` — example environment с переменными
- `README.md` — инструкция по использованию

## Содержимое

### Collections
- `collections/cerebro-sap-routes.postman_collection.json`
- `collections/raspil.postman_collection.json`
- `collections/service-delivery-svd.postman_collection.json`

### Environment
- `environments/cerebro-variable-example.postman_environment.json`

## Как использовать

1. Импортируйте нужную коллекцию в Postman
2. Импортируйте example environment
3. Выберите environment
4. Заполните переменные своими значениями
5. Выполняйте запросы

## Важно

В репозитории используются только примерные значения переменных.  
Реальные токены, ключи, cookie и другие секреты не публикуются.
